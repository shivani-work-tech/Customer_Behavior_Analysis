Customer Behavior Data Analytics Project
 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data loading in Python to delivering business insights through SQL analysis, Power BI dashboards, and executive-level reports.
The objective of this project is to analyze customer shopping behavior, uncover trends and patterns, and present insights in a clear and actionable manner suitable for stakeholders.
